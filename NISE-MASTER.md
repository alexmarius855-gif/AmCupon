> Creat: 02.07.2026. Se actualizează la fiecare nișă testată (mută-i statusul, nu șterge istoricul).

## 🔄 Circuitul semi-închis (strategia lui Alex)

```
1. DESCOPERIM nișe        → intră în nise-master.csv cu scor
2. TESTĂM prin afiliere   → pagină pe AmCupon / site mic / canal social (cost ~0)
3. MĂSURĂM                → clickuri afiliate (GA4), comisioane (2P/PS/Impact), trafic (GSC)
4. ALEGEM câștigătoarele  → scor real > scor estimat = promovare la nivel următor
5. CONSTRUIM active proprii → produs/brand/Shopify DOAR în nișe validate cu bani reali
6. PROMOVĂM prin canalele noastre → AmCupon, newsletter, social, site-uri de nișă
7. REINVESTIM              → SEO, conținut, tool-uri, automatizări
```

**Regula de aur:** o nișă avansează de la `idee → test → validat → activ propriu` doar cu
DATE (clickuri/comisioane reale), niciodată pe entuziasm. Salariul din 05.07 se alocă
doar pe nișe aflate minim la nivelul „test cu semnal pozitiv".

## 📐 Sistemul de scor (0–50 puncte)

| Criteriu | Pondere | Scală |
|----------|---------|-------|
| 💰 Potențial bani (comision × volum) | ×3 | 1–5 |
| 📈 Potențial trafic (căutări RO) | ×2 | 1–5 |
| 🔍 Ușurință SEO (5 = concurență mică) | ×2 | 1–5 |
| 🔗 Sinergie cu ce avem (AmCupon/canale/motoare) | ×2 | 1–5 |
| ❤️ Îmi place / pot susține pe termen lung | ×1 | 1–5 |

**Scor = bani×3 + trafic×2 + seo×2 + sinergie×2 + plăcere×1** → max 50.

Praguri: **≥38 = ACUM** · 30–37 = MAI TÂRZIU · <30 = DOAR IDEE.

Câmpuri per nișă (coloanele CSV): nume, categorie, public țintă, ce promovăm, programe
afiliere, comision estimat, cele 5 scoruri, cost pornire, timp până la bani, poate deveni
business propriu (D/N), site separat / newsletter / tool / produs digital / Shopify (D/N),
scor total, prioritate, status (`idee/test/validat/activ`).
